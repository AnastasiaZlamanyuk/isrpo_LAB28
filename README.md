##  Описание

Проект представляет собой простое Web API для управления коллекцией видеоигр.  
Он позволяет:
- получить список всех игр;
- получить одну игру по ID;
- добавить новую игру;
- обновить существующую игру;
- удалить игру.

## Инструкция по запуску

```bash
cd GamesApi 
dotnet run
```

## Таблица всех маршрутов

|Метод|Маршрут|Описание|Статус|
|---|---|---|---|
|**GET**|`/api/games`|Получить все игры|200|
|**GET**|`/api/games/{id}`|Получить игру по id|200/404|
|**POST**|`/api/games`|Добавить игру|201|
|**DELETE**|`/api/games/{id}`|Удалить игру|204/404|

## Примеры curl-команд

### 1. GET — получить все игры
```bash
curl http://localhost:5000/api/games
```

### 2. GET — получить одну игру
```bash
curl http://localhost:5000/api/games/1
```

### 3. POST — добавить игру
```bash
curl -X POST http://localhost:5000/api/games \
  -H "Content-Type: application/json" \
  -d "{\"title\": \"Stardew Valley\", \"genre\": \"Simulation\", \"releaseYear\": 2016}"
```

### 4. DELETE — удалить игру
```bash
curl -X DELETE http://localhost:5000/api/games/1
```
